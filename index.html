FAQ - Frequently Asked Questions (& answers)

Q: Is Premium license key lifetime?

A: Yes. Premium license key is lifetime.

Q: Is Raver Client injected?

A: Regular (Free) version is not injected, however Premium version is (harder detection).

Q: What modules does Raver Client have?

A: Raver Client includes most popular modules & some new ones, most useful are: KillAura, Reach, Speed, Step, Blink & logout spots.

Q: Can using Raver Client get me banned?

A: Yes. Using Raver Client can get you banned on non-anarchy servers where cheating and/or using cheat clients is against rules. We highly discourage to use Raver Client on non-anarchy servers. We are not responsible for punishments for using Raver Client or anything associated with it.

Q: Is Raver Client safe?

A: Yes. Raver Client is malware-free.

Q: What data does Raver Client collect?

A: Raver Client Free version does not collect any data, however the Premium one does collect as little data as possible to verify your license key is not cracked (it does collect HWID).

A: We are planning to release it within one month.
